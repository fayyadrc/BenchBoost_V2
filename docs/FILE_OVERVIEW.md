# Data Folder File Overview (Knowledge Base)
This document provides a concise summary of each file in the `data` directory.
---
## api_client.py
Handles API requests and communication with external data sources (e.g., FPL API).

## cache.py
Implements caching logic to store and retrieve data efficiently, reducing redundant API calls.

## livefpl_scrape.py
Scrapes live FPL data, likely for real-time updates or statistics.

## stats.py
Processes and analyzes FPL statistics, providing utility functions for data insights.
